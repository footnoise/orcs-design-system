# Tooltip

[Default tooltip](#default-tooltip) | [Directional variants](#directional-variants) | [Size](#size) | [On dark background](#on-dark-background) | [Properties](#properties)

This tooltip component supports customisation of direction and width. This is so that you can ensure that the tooltip doesn't run off the screen, and that the width suits the amount of text in the tooltip.

It's ok for the tooltip text to wrap if it's a long sentence or paragraph, just ensure it looks ok, i.e. no single trailing words etc.

## Default tooltip

Default tooltip is positioned at the top, and has a width of 200px.

  <Story name="defaultTooltip">{stories.defaultTooltip()}</Story>
</Preview>

## Directional variants

The direction prop allows the tooltip to be positioned in different ways to best suit how and where it is used in the UI.

## Size

Default width is 200px, can modify width to best suit the text in the tooltip. Make sure to specify units (e.g. px, %, etc.).
