# Divider

The Divider component should be used to separate distinct sections of the UI.

## Props

<Props of={Divider} />

## Examples

### Standard options

To be used throughout most of the app, in most cases purely the default Divider with no props should be used.

<Preview>
  <Story name="Basic">{stories.basicDivider()}
  </Story>
</Preview>

### Inverted options

The inverted prop should be used if a divider is required on a dark background.

<Preview>
  <Story name="Inverted">
    <Box bg="greyDarkest">
{stories.invertedDivider()}
    </Box>
  </Story>
</Preview>

### Span Grid

This prop is to be used when you need to use a divider between rows in a grid, this will allow the divider to span across all columns in the grid.

<Preview>
  <Story name="Span Grid">
  {stories.spanGrid()}
  </Story>
</Preview>
