# Badge

[Properties](#properties) | [Examples](#examples)

Badges should be used for things like status, indicators, or other additional information that you want to highlight but not make interactive/clickable.

<Preview>
  <Story name="Basic">
    <Badge>Default</Badge>
  </Story>
</Preview>

## Properties

You can use `space` and `layout` Styled-System properties with `Badge`.

### For reference, see:

- [space on Styled-System](https://styled-system.com/table#space)
- [layout on Styled-System](https://styled-system.com/table#layout)
- [systemtheme.js](https://github.com/orchestrated-io/orca-design-system/blob/master/packages/orcs-design-system/lib/systemtheme.js) (our arrays for design system values)
