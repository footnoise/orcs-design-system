# StyledLink

[Default](#default) | [Targeted](#targeted) | [Bold](#bold) | [Inverted](#inverted) | [Using Router](#using-router) | [Properties](#properties)

This `StyledLink` component supports both standard html hyperlinks and react Link components (if using react router for example).

Use `to={DESTINATION}` to use the react-router `Link` instead of the html `<a>`. If doing so, `StyledLink` will need to be enclosed in a `BrowserRouter` or similar.
