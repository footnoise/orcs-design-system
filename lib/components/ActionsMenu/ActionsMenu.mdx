# Actions Menu

[Default actions menu](#default-actions-menu) | [Left offset actions menu](#left-offset-actions-menu) | [Properties](#properties)

This component should be used when an item in the UI has multiple actions that can be taken on it.

The child components that are accepted by this component are standard html hyperlinks `<a>` or buttons `<button>`, or react router `<Link>` depending on what the function of the menu item needs to be.
