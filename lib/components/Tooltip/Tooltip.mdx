# Tooltip

[Default tooltip](#default-tooltip) | [Direction](#direction) | [Size](#size) | [Inverted](#inverted) | [Properties](#properties)

This tooltip component supports customisation of direction and width. This is so that you can ensure that the tooltip doesn't run off the screen, and that the width suits the amount of text in the tooltip.

It's ok for the tooltip text to wrap if it's a long sentence or paragraph, just ensure it looks ok, i.e. no single trailing words etc.

## Default tooltip

Default tooltip is positioned at the top, and has a width of 200px.
